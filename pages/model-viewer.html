QUICK-START EXAMPLE


Aim:
create a working web AR that will enable the users to view it through their native browsers.
The users should be able to perform basic
transformations like scale, rotate, and move the object.

Requirements:

1. 3D model

CDN 
https://cutt.ly/MLTACDt

2. <model-viewer> external script

Script 

            <script>
            
            type="module" 
            src="https://unpkg.com/@google/model-viewer/dist/model-viewer.min.js"
            
            </script>

3. <model-viewer> html snippet

Code Snippet 

            <model-viewer
            
            alt="robot model" 
            src="https://cutt.ly/MLTACDt" 
            ar 
            ar-modes="webxr scene-viewer quick-look" 
            shadow-intensity="1"
            camera-controls 
            enable-pan
            
            ></model-viewer>

Combined Code Snippet:

Complete Code 

            <!-- Import the component -->
            <script>
            
            type="module" 
            src="https://unpkg.com/@google/model-viewer/dist/model-viewer.min.js"
            
            </script>
            
            <!-- Use it like any other HTML element -->
            <model-viewer
            
            alt="robot model" 
            src="https://cutt.ly/MLTACDt" 
            ar 
            ar-modes="webxr scene-viewer quick-look" 
            shadow-intensity="1"
            camera-controls 
            enable-pan
            
            ></model-viewer>

Procedure:
1. Create an index.html file.
2. Add <model-viewer> external scripts.
3. Add <model-viewer> code snippet inside the body tag.
4. Add the above mentioned 3D asset OR add your own 3D asset to the <model-viewer> "src" property.
5. Compile and run the index.html file to view your AR.

Challenge:
Create a web AR experience using model-viewer and experiment by adding various 3D assets to it.
